# apkextractor(APK Kit)
一个Android平台用于快速导出并分享本地已安装应用的APP，支持将应用外部存储空间中的data与obb数据和apk一同打包成zip发送给对方。本工具支持模糊搜索、多种排序方式来快速定位和查找应用。
